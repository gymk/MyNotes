### Node.js - What it is

- It is a JavaScript runtime built on Chrome's V8 JavaScript engine
- Node.js uses an event-drive, non-blocking I/O model to achieve efficiency while remaining lightweight

### Features of Node.js

- V8
- Asynchronous
- Single-threaded
- Unified API
- Data Streams
